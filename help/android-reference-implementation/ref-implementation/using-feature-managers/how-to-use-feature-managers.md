# フィーチャマネージャの使い方{#how-to-use-feature-managers}

機能マネージャを使用して、機能の有効化、無効化、設定を行う方法は2つあります。

* 設定情報を設定ファイルからメディアプレイヤーに渡すことで、機能マネージャーを作成および設定できます。
* マネージャーファクトリを使用して、コードを調べずに機能を有効または無効にできます。
