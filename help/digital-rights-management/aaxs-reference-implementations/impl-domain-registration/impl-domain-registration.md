# ドメイン登録の実装{#implementing-domain-registration}

リファレンス実装ライセンスサーバは、ドメイン登録ビジネスロジックの2つのバリエーションを示します。 一般的なドメインサーバーでは、次のワークフローのうち1つだけが実装されます。

* IDベースのドメイン
* 匿名ドメイン
