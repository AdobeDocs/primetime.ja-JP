# デバッグとログ{#debugging-and-logging}

ブラウザーTVSDKは、デフォルトでログを有効にしています。 また、SDKからのログを有効または無効にするAPIのサポートも提供します。 AdobePSDK.LogManager.enableLogging APIを使用して、ログを有効または無効にします。
