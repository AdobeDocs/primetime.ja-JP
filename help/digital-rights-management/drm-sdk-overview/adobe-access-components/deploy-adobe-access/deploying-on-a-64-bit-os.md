# 64ビットオペレーティングシステムへの展開{#deploying-on-a-bit-operating-system}

64ビット版のRedHatやWindowsなどの64ビット版のオペレーティングシステムは、32ビット版のオペレーティングシステムよりもはるかに高いパフォーマンスを提供します。
