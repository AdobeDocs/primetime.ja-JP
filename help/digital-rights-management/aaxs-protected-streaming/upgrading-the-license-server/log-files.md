# ログファイル{#log-files}

保護されたストリーミングアプリケーション用にAdobe Access Serverが生成するログファイルは、*LicenseServer.LogRoot*&#x200B;で指定されたディレクトリにあります。

>[!NOTE]
>
>サーバーの実行中に現在のログファイルを削除または移動した場合、ログファイルは再作成されず、一部のログ情報が失われます。
