# Media Packager {#using-media-packager}の使用

Media Packagerを使用して、暗号化するファイル内のデータと、コンテンツファイルに適用するポリシーを指定できます。 例えば、ビデオデータは暗号化するが、オーディオデータは暗号化しないように指定できます。
