# テナントログファイル{#tenant-log-file}

各テナントのテナントログファイルは、 [!DNL flashaccess-tenant.log]LicenseServer.LogRoot *tenantnameにあ* ります [!DNL /flashaccesserver/tenants/]**。 テナントログには、このテナントに対して生成された各ライセンスを説明する監査情報が含まれます。
