# 概要{#overview}

TVSDK Primetime Reference Androidプロジェクトは、Eclipseとの互換性を維持するために、大幅な設定やIDEの変更を行わないで設定されます。 Primetime Referenceプロジェクトをセットアップするには、まずEclipseを設定する手順に従ってから、プロジェクトを構築する手順に従います。
