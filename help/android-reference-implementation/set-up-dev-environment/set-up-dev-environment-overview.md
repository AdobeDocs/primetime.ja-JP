# 概要 {#overview}

TVSDK Primetime Reference Android プロジェクトは、大きな設定や IDE の変更を行わずに、Eclipse と互換性を持つように設定されます。 Primetime Reference プロジェクトを設定するには、まず手順に従って Eclipse を設定し、次に手順に従ってプロジェクトを構築します。
