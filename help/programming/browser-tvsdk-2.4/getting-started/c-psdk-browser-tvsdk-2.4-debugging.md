# デバッグとログ{#debugging-and-logging}

ブラウザーTVSDKは、デフォルトでログを有効にしています。 また、SDKからのログの有効/無効を切り替えるAPIのサポートも提供します。 AdobePSDK.LogManager.enableLogging APIを使用して、ログを有効または無効にします。
