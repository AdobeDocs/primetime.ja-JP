# 要件 {#requirements}

次のファイルがインストールされていることを確認する必要があります。

* Tomcat 6.0以降
* MySQLなどのデータベース（License Serverコンポーネントにのみ必要）
* Java 1.6以降
* Ant 1.8以降（サンプルのビルドスクリプトを使用）

TomcatとMySQLをインストールしたら、アドビから資格情報を取得します。
