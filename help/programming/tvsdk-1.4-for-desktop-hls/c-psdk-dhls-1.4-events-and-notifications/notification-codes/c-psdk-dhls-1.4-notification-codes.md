# 概要{#notification-codes-overview}

TVSDK通知システムは、診断メタデータを提供する様々なエラー、警告および情報通知を生成します。

通知オブジェクトは、プレイヤーのステータスに関する情報を提供します。 TVSDKは、時系列にソートされた通知オブジェクトのリストを提供し、各通知には次のメタデータが含まれます。
