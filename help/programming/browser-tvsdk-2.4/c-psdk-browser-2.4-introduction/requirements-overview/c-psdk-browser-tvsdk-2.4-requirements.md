# 概要{#requirements-overview}

ブラウザーTVSDKには、メディアコンテンツ、マニフェストコンテンツ、ソフトウェアバージョンに固有のプロパティが必要です。

ブラウザーTVSDKリリースには、次のプレーヤーが含まれています。

* `Primetime.js` （デスクトップおよびモバイルでのHLSの場合）

プレーヤーには、次の参照APIが含まれます。

* `Primetime.js` ブラウザーTVSDK
* コア再生用JavaScript API
* デモプレーヤーの参照

      * Chrome 33+

      * Edge+
