# ログファイル{#log-files}

保護されたストリーミングアプリケーション用にAdobe Access Serverで生成されたログファイルは、 *LicenseServer.LogRoot*.

>[!NOTE]
>
>サーバーの実行中に現在のログファイルが削除または移動された場合、ログファイルが再作成されない可能性があり、一部のログ情報は失われます。
