# SSL の設定 {#configuring-ssl}

Secure Sockets Layer(SSL) は、FlashMediaRights Managementサーバー 1.x クライアントをサポートする必要がある場合にのみ必要です。 クライアント認証を使用する SSL は、Adobeアクセスキーサーバーに必要です。 詳しくは、「Adobeアクセスキーサーバーの使用」を参照してください。
