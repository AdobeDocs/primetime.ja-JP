# 要件{#requirements}

ビルド環境は、次の最小要件を満たす必要があります。

* Microsoft Windows Server 2008またはRed Hat® Enterprise Linux® 5.6
* OracleJava JRE 1.6(カスタム認証拡張の作成にはOracleJava JDK 1.6が必要)
* DVDの[!DNL Third Party\Tomcat\6.0.18]フォルダーにあるApache Tomcat® 6
* Adobeが発行した資格情報
