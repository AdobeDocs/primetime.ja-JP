# Primetimeリファレンスの実装の使用方法 {#how-to-use-the-primetime-reference-implementation}

Primetimeリファレンスの実装は、モジュール式のプレーヤーで、特殊な機能マネージャーを通じて簡単に変更できる個々の機能に分類されています。 これらの機能マネージャーは、アプリケーションとTVSDKライブラリを接続するブリッジとして使用されます。

参照実装は、次の方法で使用できます。

* コードを変更せずに、そのまま使用します（すべての機能がデフォルト値で有効になっています）。
* TVSDKライブラリの使用方法を理解するためのリファレンスとして使用します。
* 使用しない機能をオフにして、アプリケーションに適用する機能を選択します。
* 機能を変更せずに、UIコンポーネントをカスタマイズします。

Primetimeリファレンスの実装を提供し、TVSDKを理解し、機能マネージャーを簡単に変更してプレイヤーをカスタマイズできるようにします。 ただし、TVSDKライブラリに関する詳細は、 [TVSDK 1.4 for Androidプログラマーガイドを参照してください](https://helpx.adobe.com/content/dam/help/en/primetime/programming-guides/psdk_android.pdf) 。

Javadoc形式のリファレンス実装APIドキュメントに簡単にアクセスするには、ここをクリック [します](https://help.adobe.com/en_US/primetime/api/reference_implementation/android/javadoc/index.html)。
