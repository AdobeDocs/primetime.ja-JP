# 要件 {#requirements}

以下がインストールされていることを確認する必要があります。

* Tomcat 6.0 以降
* MySQL などのデータベース（License Server コンポーネントにのみ必要）
* Java 1.6 以降
* Ant 1.8 以降（サンプルビルドスクリプトを使用）

Tomcat と MySQL をインストールしたら、Adobeから資格情報を取得します。
