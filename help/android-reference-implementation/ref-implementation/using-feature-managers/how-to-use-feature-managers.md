# 機能マネージャの使用方法 {#how-to-use-feature-managers}

機能マネージャを使用して、機能の有効化、無効化、設定を行う方法は 2 つあります。

* 設定ファイルからメディアプレーヤーに設定情報を渡すことで、機能マネージャーを作成して設定できます。
* マネージャーファクトリを使用すると、コードを調べずに機能を有効または無効にできます。
