# Crossdomain ポリシーファイル {#crossdomain-policy-file}

License Server からFlashRuntime クライアントにライセンスを要求するには、クロスドメインポリシーファイルが必要です。 詳しくは、 *コンテンツの保護にAdobeアクセス SDK を使用する* を参照してください。
