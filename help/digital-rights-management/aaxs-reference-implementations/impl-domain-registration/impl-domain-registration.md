# ドメイン登録の実装 {#implementing-domain-registration}

参照実装ライセンスサーバは、ドメイン登録ビジネスロジックに関する2つのバリエーションを示します。 一般的なドメインサーバーでは、次のワークフローの1つのみが実装されます。

* IDベースのドメイン
* 匿名ドメイン
