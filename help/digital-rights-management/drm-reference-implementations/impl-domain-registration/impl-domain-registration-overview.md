# 概要{#implement-domain-registration-overview}

リファレンス実装ライセンスサーバーでは、ドメイン登録ビジネスロジックの2つのバリエーションが示されます。

* **IDベース**  — 認証された各ユーザーは、ドメインと、そのドメインに参加できる特定の数のデバイスを持ちます。
* **匿名**  — 多数のデバイスが1つのドメインに属しており、認証が必要でない場合があります。
