# デバッグとログ{#debugging-and-logging}

ブラウザー TVSDK は、デフォルトでログを有効にしています。 また、SDK からのログを有効または無効にする API サポートも提供されます。 ログを有効または無効にするには、AdobePSDK.LogManager.enableLogging API を使用します。
