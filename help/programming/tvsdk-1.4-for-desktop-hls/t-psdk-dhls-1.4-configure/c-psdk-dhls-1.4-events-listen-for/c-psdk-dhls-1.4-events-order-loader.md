# ローダイベント{#loader-events}

プレイヤーは、次のイベントに基づいてアクションを実装できます。

| イベント | 意味 |
|---|---|
| `MediaPlayerItemLoader.completed` | メディアリソースの読み込みが正常に完了しました。 |
| `MediaPlayerItemLoader.failed` | メディアリソースの読み込みで問題が発生しました。 |
