# テナントログファイル{#tenant-log-file}

各テナントのテナントログファイル [!DNL flashaccess-tenant.log]は、次の場所にあります。 *LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*tenantname*. テナントログには、このテナントに対して生成された各ライセンスを説明する監査情報が含まれます。
