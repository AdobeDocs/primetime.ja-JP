# ダウンロードして所有 {#download-to-own}

ダウンロード先(DTO)の使用モデルを使用すると、ユーザーはコンテンツをダウンロードしてオンラインまたはオフラインで使用し、コンテンツの永久的なライセンスを発行できます。 ライセンスを要求する場合、ユーザーは認証を受け、ユーザーがコンテンツを購入したかどうかをサーバーで確認できるようにする必要があります。
