# テナントログファイル{#tenant-log-file}

各テナントのテナントログファイル[!DNL flashaccess-tenant.log]は、*LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*tenantname*&#x200B;にあります。 テナントログには、このテナントに対して生成された各ライセンスを説明する監査情報が含まれます。
