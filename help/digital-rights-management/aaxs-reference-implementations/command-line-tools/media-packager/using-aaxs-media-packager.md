# Media Packagerの使用 {#using-media-packager}

Media Packagerを使用して、暗号化するファイル内のデータと、コンテンツファイルに適用するポリシーを指定できます。 例えば、ビデオデータは暗号化され、オーディオデータは暗号化されないように指定できます。
