条件：2022 年 3 月 02 日水曜日午前 1:30 ～ 4:30 東部時間

* 広告クリエイティブ呼び出し、広告サーバー呼び出し、Avail 解決イベントを含む、新しい広告パフォーマンスイベントのキャプチャ。
