# Adobe Primetime仕様

Adobe Primetimeは、IP 接続されたすべての画面に TV を提供します。 プログラマーやオペレーターに、デスクトップやデバイスにわたるビデオのストリーミング、保護、分析、収益化をおこなうモジュラー機能を提供します。 これらのエクスペリエンスを強化するには、Primetime ビデオプレーヤーにコンテンツや広告を配信するために使用される、多数のストリーミングテクノロジーが使用されます。

Adobe Primetime Digital Program Insertion Specification（バージョン 1.2）では、Primetime ストリームに広告や代替コンテンツを挿入する際に使用されるシグナリングを定義しています。 この仕様では、RTMP、HDS、HLS に使用されるシグナリングを定義します。

## Adobe Primetime Digital Program Insertion Specification（バージョン 1.2） {#digital-program-insertion-specification}

Adobe Primetime Digital Program Insertion Specification（バージョン 1.2）では、Primetime 互換の RTMP、HDS、HLS ストリームに広告や代替コンテンツを挿入する際に使用するシグナリングを定義しています。

[Primetime デジタルプログラム挿入仕様のダウンロード](assets/PrimetimeDigitalProgramInsertionSignalingSpecification.pdf)

## Adobe Primetime HTTP Live Streaming Profile（2014 年 4 月） {#http-live-streaming-profile}

Adobe Primetime HTTP Live Streaming Profile(PT-HLS) は、HTTP Live Streaming(HLS) プロトコルの拡張機能のセットで、Adobeアクセスで保護されたコンテンツ、広告コンテンツ、広告シグナリングなどのプレミアムビデオの使用例を有効にします。

[Adobe Primetime HTTP Live Streaming プロファイル仕様のダウンロード](assets/PrimetimeHLS_April2014.pdf)

## Adobe Primetime Dynamic Adaptive Streaming over HTTP(DASH)Interoperability Guidelines（2016 年 1 月） {#dynamic-adaptive-streaming-http}

このドキュメントでは、Adobe Primetimeと互換性のある DASH コンテンツを作成するための要件について説明します。

[HTTP(DASH) 相互運用性に関するガイドライン「Adobe Primetime Dynamic Adaptive Streaming over HTTP(DASH)」のダウンロード](assets/PrimetimeDASH_Jan2016.pdf)
