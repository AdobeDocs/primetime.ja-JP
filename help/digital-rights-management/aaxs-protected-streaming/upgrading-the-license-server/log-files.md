# ログファイル{#log-files}

保護されたストリーミングアプリケーション用にAdobe Access Serverで生成されたログファイルは、 *LicenseServer.LogRootで指定されたディレクトリに格納されます*。

>[!NOTE] {class=&quot;- topic/note &quot;}
>
>サーバーの実行中に現在のログファイルが削除または移動された場合、ログファイルは再作成されず、一部のログ情報が失われます。
