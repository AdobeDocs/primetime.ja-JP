# 要件{#requirements}

ビルド環境は、次の最小要件を満たす必要があります。

* Microsoft Windows Server 2008またはRed Hat® Enterprise Linux® 5.6
* Oracle Java JRE 1.6（カスタム認証拡張の作成にはOracle Java JDK 1.6が必要）
* DVDのフォルダにあるApache Tomcat® 6 [!DNL Third Party\Tomcat\6.0.18] を参照
* アドビが発行した資格情報
