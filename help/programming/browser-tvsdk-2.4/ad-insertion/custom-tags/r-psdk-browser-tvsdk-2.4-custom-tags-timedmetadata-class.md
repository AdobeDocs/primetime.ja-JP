# 時間指定メタデータクラス{#timed-metadata-class}

ブラウザーTVSDKは、プレイリスト/マニフェスト内にサブスクライブされたタグを検出すると、タグを自動的に処理して、TimedMetadataオブジェクトとして公開しようとします。

`TimedMetadata`クラスは次の要素を提供します。
