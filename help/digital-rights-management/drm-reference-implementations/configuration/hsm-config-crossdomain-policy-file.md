# クロスドメインポリシーファイル{#cross-domain-policy-file}

Flashランタイムクライアントがライセンスサーバーからライセンスをリクエストする必要がある場合は、クロスドメイン DRM ポリシーファイルが必要です。 クロスドメインポリシーについて詳しくは、 *コンテンツの保護にAdobe Primetime DRM SDK を使用*.
