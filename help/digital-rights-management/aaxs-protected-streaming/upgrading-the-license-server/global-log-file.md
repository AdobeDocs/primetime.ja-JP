# グローバルログファイル{#global-log-file}

グローバルログファイルflashaccess-global.logは、*LicenseServer.LogRoot*&#x200B;にあります。 このログには、AdobeアクセスSDKによって生成されたログメッセージや、サーバーの初期化時に生成されたログメッセージを含めることができます。
